Drop your project screenshots here. The portfolio will automatically use them.

## Tips:
- Recommended size: 1280×720px (16:9) or 800×500px
- PNG or JPG both work
- The image will be cropped to fill the card — landscape screenshots work best
- Until you add images, a styled gradient placeholder with the project icon is shown automatically
